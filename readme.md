#  Sistema de Gestión de Streaming Justin Salazar - Autonomos

Este es un sistema de gestión de usuarios y videos en el cual utilizamos Go

##  Funcionalidades Principales
- Registro de usuarios 
- Contraseña encriptada 
- Creación de videos con un nombre unico 
- Encapsulación y getters
- Uso de interfaces para gestionar
- Lista de videos en memoria

## Estructura del proyecto

- `models/` → Usuarios y videos
- `services/` → Interfaz GestorContenido
- `utils/` → Contraseña hash
- `main.go` → Punto de entrada y pruebas

## Lo vamos a ejecutar de la suguiente forma en la terminal


go run main.go
